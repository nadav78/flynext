## STUDENT 1: USER AUTHENTICATION & ACCOUNT MANAGEMENT

### Responsibilities:
- Implement user registration, login, logout functionality
- Design and implement JWT authentication system
- Develop user profile management (create, read, update)
- Set up authorization middleware for protected routes

### Database Models:
- User model (first/last name, email, password, phone, profile picture)
- Session/Token management

### API Endpoints:
- POST /api/users/register
- POST /api/users/login
- POST /api/users/logout  
- GET /api/users/profile
- PUT /api/users/profile
- POST /api/users/refresh-token

## STUDENT 2 (Robert): FLIGHT SEARCH SYSTEM & ADVANCED FLIGHTS SYSTEM INTEGRATION

### Responsibilities:
- Implement integration with external Advanced Flights System (AFS)
- Create secure proxies for AFS API calls through the backend
- Implement flight search functionality with filtering options
- Develop airport/city autocomplete feature
- Create flight details display API

### API Endpoints:
- GET /api/flights/search (query parameters: source, destination, date, trip-type)
- GET /api/airports/autocomplete (query parameter: search term)
- GET /api/flights/:id (detailed flight information)
- POST /api/flights/book (book a flight)
- GET /api/flights/verify/:bookingId (verify flight booking)

## STUDENT 3: HOTEL MANAGEMENT & BOOKING SYSTEM

### Responsibilities:
- Implement hotel registration and management
- Develop room type definition and management
- Create booking management system
- Implement availability tracking and modification

### Database Models:
- Hotel model (name, logo, address, location, star-rating, images)
- Room Types model (name, amenities, price, images)
- Bookings model
- Availability model

### API Endpoints:
- POST /api/hotels (create hotel)
- GET /api/hotels/:id (hotel details)
- POST /api/hotels/:id/room-types (add room type)
- GET /api/hotels/:id/room-types
- GET /api/hotels/search (with filtering by location, dates, etc.)
- GET /api/hotels/:id/bookings (with filtering)
- POST /api/hotels/:id/bookings/:bookingId/cancel
- GET /api/hotels/:id/availability
- PUT /api/hotels/:id/room-types/:roomTypeId/availability
